=== FINDING 5 — CUSTOM CURSOR HIDES DEFAULT CURSOR (SEVERITY: HIGH) ===
Affects: BOTH pages + entire site
Global CSS rule:  *, *::before, *::after { cursor: none !important; }
This disables the system cursor sitewide and replaces it with a custom
image-based cursor (.cursor-wrapper). On mobile, tablets, and any device
where JS fails to load or where the custom cursor animation breaks,
users have NO visible cursor at all.

Selector: embedded <style> in <head>
FIX: Remove "cursor: none !important" or scope it to desktop-only via
@media (hover: hover) and (pointer: fine).
